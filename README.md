# 🚀 Nexus AI - Modern Software Landing Page

A futuristic and responsive landing page for an AI platform, focused on modern design (Glassmorphism) and high performance.

## ✨ Demo
🔗 **[Live Demo](https://nexus-ai-software.vercel.app/)**

## 🛠️ Technologies Used
* **HTML5** - Semantic structure.
* **CSS3** - Responsive design, CSS Variables, and Glassmorphism effects.
* **JavaScript** - DOM manipulation for mobile navigation.
* **Vercel** - Deployment and Hosting.

## 📱 Features
- [x] Fully Responsive Design (Mobile & Desktop).
- [x] Interactive Mobile Menu with auto-close functionality.
- [x] Modern Glassmorphism card effects.
- [x] Smooth scrolling between sections.
- [x] Clean and optimized code.
